## validate.py



Required arguments:

-i/--input_file INPUT_FILE : name of the input file

-n/--num_rand_sent NUM_RANDOM : Number of randomly fetched sentences






To run the code:

```
$ python validate.py -i INPUT_FILE.* -n NUM_RANDOM

```


Example:


```
$ python validate.py -i ./AN-BLO_df -n 20

```
